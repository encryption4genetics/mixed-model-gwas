# mixed.model.gwas

Run a MM GWA.

## INSTALL

    git clone git@github.com:encryption4genetics/mixed-model-gwas.git
    R CMD INSTALL mixed-model-gwas

Fire up R and

    R
    library(mixed.model.gwas)
